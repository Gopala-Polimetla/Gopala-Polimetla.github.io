### GitHub Commands

*git init*

*git branch*

*git branch [branch name]*

*git checkout*

*git status*

*git add [file]*

*git add .*

*git commit -am "Your Message"*

*git push origin [branch name]*

*git pull*

*git clone [URL]*

*git config --global user.name "[username]"*

*git config --global user.email "[your email]"*
